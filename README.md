# Akai AX 80

The Akai AX-80 is an aggressive sounding CEM based polysynth from the early 80s. It looks pretty cool too.


-   **Formats:** EXS24, Kontakt & SFZ
-   **Info:** [Wikipedia](https://en.wikipedia.org/wiki/Akai_AX80)
